## Example

示例中包含了

- 一个简单的Hello World示例演示如何使用组件替代Trial
- [Simple Reaction Time Task](https://www.jspsych.org/v7/tutorials/rt-task/) 演示了如何在JsPsych-Vue使用JsPsych的Plugin.

如果你想运行示例, 首先克隆整个仓库

```shell
git clone https://github.com/HGGshiwo/jspsych-vue.git
```

在根目录中安装依赖

```
yarn install
```

启动服务

```
yarn serve
```
